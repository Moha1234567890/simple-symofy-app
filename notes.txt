




for installing twig => the template folder will be generated
composer require twig


for installing doctrine 
composer require doctrine maker

change your credintials(username, db name, password) in the .env file 

and run php bin/console doctrine:database:create   to create your db


to create an entity => table  

php bin/console make:entity Article

create and run migrations
 php bin/console make:migration
 php bin/console doctrine:migrations:migrate
